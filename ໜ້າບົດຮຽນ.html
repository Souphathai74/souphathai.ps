    ເປັນນາມສະກຸນໄຟລ໌ທີ່ເໝາະກັບຮູບທີ່ມີເສດທີ່ຊັບຊ້ອນເຊັ່ນ: ຮູບການ, ຕົວອັກສອນ, ຮູບລາຍ ເສັ້ນຕ່າງໆ ຈຸດເດັ່ນຂອງຮູບພາບຊະນິດນີ້ຄື ເມື່ອນຳມາໃຊ້ກັບຮູບທີ່ມີເສດບໍ່ຫຼາຍຈະມີຂະໜາດໄຟລ໌ນ້ອຍ ໂຫຼດ ສະແດງຜົນໄດ້ໄວ ຈຶ່ງນິຍົມໃຊ້ສ້າງເປັນພາບເຄື່ອນໄຫວ (Animation) ປະກອບເວບໄຊທ໌, ໄອຄ່ອນເວບ, ແຖບ ເມນູ, ປ້າຍໂຕສະນາເວບ ເປັນຕົ້ນ ແລະ ອີກຈຸດເດັ່ນໜຶ່ງທີ່ຮູບພາບນາມສະກຸນ GIF ສາມາດເຮັດໄດ້ນັ້ນກໍ່ຄື ການ ບັນບັນທຶກຮູບພາບໂດຍໃຫ້ພື້ນຫຼັງກາຍເປັນແບບຫວ່າງເປົ່າ ຫຼື ພື້ນຫຼັງໃສໄດ້.

    1.3.3 ປະເພດໄຟລ໌ PNG (Portable Network Graphic)

    ເປັນນາມສະກຸນຂອງໄຟລ໌ທີ່ພັດທະນາມາຈາກໄຟລ໌ GIF ໂດຍມີຈຸດເດັ່ນທີ່ມີເສດສີຫຼາຍ ແລະ ຄວາມຊັດເຈນລະອຽດສູງກວ່າແບບ jpg (ແຕ່ກໍ່ມີຂະໜາດໄຟລ໌ທີ່ໃຫຍ່ຂຶ້ນເຊັ່ນດຽວກັນ) ໂດຍຈະມີລະດັບຄວາມ ລະອຽດຂອງເສດສີໃຫ້ເລືອກ (ລະອຽດຫຼາຍຊ່ຳໃດ ແມ່ນເຮັດໃຫ້ເນື້ອທີ່ໄຟລ໌ຫຼາຍເຊັ່ນກັນ) ໃນຂະນະທີ່ບັນທຶກ ໄຟລໍສາມາດເລືອກໄດ້ວ່າຈະບັນທຶກແບບບໍ່ຫຼຸດຄວາມລະອຽດລົງ ຫຼື ຈະໃຫ້ຫຼຸດຄຸນນະພາບລົງກໍ່ໄດ້ ນອກຈາກນີ້ ນາມສະກຸນ PNG ຍັງນຳຈຸດເດັ່ນຂອງ GIF ມາພ້ອມ ເຊິ່ງນັ້ນໝາຍເຖິງສາມາດບັນທຶກຮູບພາບໃຫ້ເປັນແບບພື້ນ ຫຼັງໃສ ຫຼື ໃສແບບບໍ່ສົມບູນໄດ້.

    1.3.4 ປະເພດໄຟລ໌ TIF,TIFF (Tagged Imae File Format)

    ເປັນນາມສະກຸນໄຟລ໌ຮູບພາບ ທີ່ນິຍົມນຳມາໃຊ້ດ້ານສື່ສິ່ງພິມ ໂດຍມັກຈະເອີ້ນໃຊ້ງານໃນໂປຣ ແກຣມຈັດໜ້າໜັງສື PagcMaker ແລະ Indesirn ເພາະຮູບພາຍມີຄວາມຄົມຊັດສູງ (ຂະໜາດໄຟລໍຂ້ອນຂ້າງ ໃຫຍ່) ສາມາດເລືອກການບັນທຶກຮູບພາບໃຫ້ເປັນໄດ້ຫຼາຍແບບ ເພື່ອຮັກສາຄຸນນະພາບຂອງຮູບໃຫ້ເທົ່າເດີມ ຫຼື ຈະຫຼຸດຄຸນນະພາບຂອງຮູບພາບລົງກໍ່ໄດ້.

    1.3.5 ປະເພດໄຟລ໌ Ran File

    ແມ່ນໄຟລ໌ຂໍ້ມູນດິບທີ່ຕ້ອງຖ່າຍຮູບດີຈິຕອນເກັບບັນທຶກໄວ້ ມີຂະໜາດໄຟລ໌ໃຫຍ່ຫຼາຍ ແລະ ມີ ເສດສີຫຼາຍກວ່າໄຟລ໌ຮູບນາມສະກຸນອື່ນ ກ້ອງແຕ່ລະລຸ້ນຈະບັນທຶກສ້າງໄຟລ໌ນາມສະກຸນ Rw ແຕກຕ່າງກັນ ແລະ ໄຟລ໌ຮູບພາບຊະນິດນີ້ຈະບໍ່ສາມາດເປີດແບບປົກະຕິຄືຮູບພາບນາມສະກຸນອື່ນໄດ້ ເຊິ່ງຕ້ອງມີໂປຣແກຣມສະເພາະທີ່ ມີຄຸນສົມບັດໃນການເປີດເບິ່ງຮູບນາມສະກຸນ Raw ໂດຍກົງ.

    1.3.6 ນາມສະກຸນປະເພດໄຟລ໌ PSD

    ເປັນໄຟລ໌ຂອງ ໂປຣແກຣມ Photoshop ທີ່ສາມາດແກ້ໄຂໄດ້.
